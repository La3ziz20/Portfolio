# Portfolio de Mohamed Aziz Manai

Bienvenue sur le code source de mon portfolio personnel ! 🚀

## Présentation

Ce projet est un site web personnel (portfolio) développé pour présenter mes compétences, mon parcours, et mes projets en tant qu'étudiant en cycle d'ingénierie informatique spécialisé en **Data Science & Intelligence Artificielle**.

Le site intègre un design moderne "Dark Mode" avec des effets de "glassmorphism" et des animations fluides au défilement.

## Fonctionnalités

*   **Design Premium** : Interface sombre moderne avec accents violets/bleus.
*   **Animations Fluides** : Éléments apparaissant doucement lors du défilement (IntersectionObserver).
*   **Totalement Responsive** : Parfaitement adapté pour mobile, tablette et desktop.
*   **Sections Incluses** :
    *   À propos / Parcours
    *   Expérience Professionnelle
    *   Projets (avec liens GitHub)
    *   Compétences & Certifications
    *   Contact

## Technologies Utilisées

*   **HTML5** : Structure sémantique du site.
*   **CSS3 (Vanilla)** : Variables CSS, Flexbox/Grid, Animations, et effets "Glassmorphism".
*   **JavaScript (Vanilla)** : Logique d'animation au défilement et menu mobile.
*   **Font Awesome** : Icônes.
*   **Google Fonts** : Polices *Outfit* et *Inter*.

## Comment exécuter le projet localement

Puisqu'il s'agit d'un projet web basique avec des fichiers statiques, vous pouvez l'ouvrir directement dans votre navigateur :

1.  Clonez ce dépôt :
    ```bash
    git clone https://github.com/La3ziz20/portfolio-.git
    ```
2.  Accédez au dossier :
    ```bash
    cd portfolio-
    ```
3.  Ouvrez le fichier `index.html` dans n'importe quel navigateur moderne, ou utilisez un serveur local (ex: `npx serve .` ou `python -m http.server 8080`).
